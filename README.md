# Bolyhos Könyvelő Iroda – weboldal

Statikus, reszponzív bemutatkozó oldal. Az `index.html` fájl közvetlenül megnyitható böngészőben.

## Tartalom

- `index.html` – szerkezet és magyar nyelvű tartalom
- `styles.css` – reszponzív megjelenés és animációk
- `script.js` – mobil menü, GYIK, beúszó elemek és kapcsolatfelvételi űrlap

## Fontos a publikálás előtt

Az iroda e-mail-címe nem volt megbízhatóan elérhető nyilvános forrásból. A kapcsolatfelvételi űrlap ezért jelenleg a látogató levelezőprogramját nyitja meg címzett nélkül. Publikálás előtt érdemes megadni a fogadó e-mail-címet vagy bekötni egy űrlapkezelő szolgáltatást.
